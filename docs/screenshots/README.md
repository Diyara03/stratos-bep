## Design System

- Theme: Light (Stratos design system)
- Sidebar: #1E3A5F (navy) — only dark element
- Sidebar sections (Phase 7): MONITOR, SECURITY, REPORTS, ADMIN
- Content background: #F8FAFC
- Cards: #FFFFFF with #E2E8F0 border
- Verdicts: CLEAN (green), SUSPICIOUS (amber), MALICIOUS (red)
- Font: Inter (body), JetBrains Mono (code/hashes)
- Resolution: 1280x800 (Figs 1-8), 1440x900 (Fig 9, 10-20)
