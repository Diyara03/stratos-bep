Admin Quick Reference

DO

  • Use a Web Application OAuth client type in Google Cloud Console
  • Set the redirect URI to match your server URL exactly
  • Test API keys after entering them
  • Keep thresholds at defaults (25/70) unless you have a reason to change
  • Monitor the dashboard for unusual detection patterns after changes

DO NOT

  • Share API keys or OAuth credentials with anyone
  • Set the clean threshold higher than malicious threshold
  • Set fetch interval below 5 seconds (rate limiting risk)
  • Disconnect Gmail during an active email analysis
  • Upload a Desktop Application credentials file (it will not work for web OAuth)
